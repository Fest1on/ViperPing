**ViperPing** is an interactive console-based application written in C++ for real-time network monitoring and host availability checks.

---

## 🚀 Features

- 🖥️ Interactive Console UI (not just CLI)
- 🌐 Ping IP addresses and domain names
- 📊 Real-time latency display
- 🔁 Continuous monitoring
- ⚡ High performance (C++)

---

## 🎮 Interface

After launching, the application opens an interactive console interface where you can:

- enter a host address  
- start / stop pinging  
- monitor results in real time  

> 💡 Unlike traditional tools, this is not a plain CLI — it provides a UI experience inside the terminal.
